Houdini内置属性Attributes

Posted on 2019-08-06 In Houdini

在Houdini中写VEX语言的时候，不可避免得需要访问Houdini中很多内置的属性Attributes，之前在看教程的时候，作者整理了一份很详细的属性列表，故搬运至此，方便以后查询。

Available Datatypes

Types Syntax
float f@name
vector2(2 floats) u@name
vector(3 floats) v@name
vector4(4 floats) p@name
int i@name
matrix2(2x2 floats) 2@name
matrix3(3x3 floats) 3@name
matrix(4x4 floats) 4@name
string s@name

General Attributes

Types Attributes Descriptions
int @ptnum Point Number
int @numpt Total number of points
int @primnum Primitive Number
int @numprim Total number of primitives
int @vtxnum Vertex Number
int @numvtx Total number of vertices
float @Time Current Time, in seconds
float @TimeInc Time increment per frame in seconds
float @Frame Current frame

Geometry Attributes

Types Attributes Descriptions
vector @P Point/Primitive Position
vector @N Point/Primitive/Vertex Normal
vector @v Velocity (coule be used for motion blur or in particle systems)
float @pscale Uniform scale. Used in copy-sop or particle systems
vector @scale Non-uniform scale
vector @up Up vector. Used together with @N to orient point/particle/instance
vector4 @orient Quaternion defining the rotation of a point/particle/instance
vector4 @rot Quaternion defining additional rotation
vecor @trans Translation of instance
matrix @transform Transformation matrix (used in copy-sop)
vecor @pivot Local pivot point for instance
float @lod Detail/primitive level of detail
vecor @rest Rest position
vecor @force Force (acting on particle)
float @age Particle age
float @life Max. Particle Life

Volumes Attributes

Types Attributes Descriptions
float @density Density of voxel
int @ix, @iy, @iz Voxel indices along each axis, ranging from 0 to resolution-1
vector @center Center of current volume
vector @orig Bottom left corner of current volume
vector @size Size of current volume
vector @dPdx Change in position to get from one voxel to the next in x direction
vector @dPdy Change in position to get from one voxel to the next in y direction
vector @dPdz Change in position to get from one voxel to the next in z direction
vector @BB Relative position inside bounding box, ranging from {0,0,0} to {1,1,1}

Shading Attributes

Types Attributes Descriptions
vector @Cd Diffuse Color
float @Alpha Alpha transparency

vector @Cs Specular Color
vector @Cr Reflective Color
vector @Ct Transmissive Color
vector @Ce Emissive Color
float @rough Roughness
float @fresnel Fresnel coefficient
float @shadow Shadow Intensity
float @sbias Shadow Bias
